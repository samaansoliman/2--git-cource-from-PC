    To Uplode project from pc to github follow this steps

  • 1- finish your project on pc
  • 2- go to github and Creating new repository
  • 3- open your Terminal And write This Command
    1. git init
    2. git add * || the file only
    3. git commit -m "Creating your Messages"
    4. git remote add origin git@github.com:samaansoliman/2--git-cource-from-PC.git [ Here we Take url ssh]
    5. git push -u origin master || main
  • Responce:- Enter passphrase for key '/c/Users/DELL/.ssh/id_rsa': [Write your SSH Key Password]
